# Codex of History v3.1.1

Статическое историческое приложение для GitHub Pages. Контент хранится в JSON, прогресс — в `localStorage`.


## v3.1.1 — исправление изображений и старта новой игры

- восстановлены 42 исторических изображения Wikimedia Commons для римских карточек; локальные SVG остаются офлайн-резервом;
- новые профили больше не получают Ромула, Рема и основание Рима до выбора кампании;
- первая коллекция создаётся из трёх карточек первой миссии выбранной кампании;
- исправлена миграция профилей, созданных с римскими стартовыми картами в другой кампании;
- полный сброс снова запускает выбор эпохи и кампании.

## v3.1 — Вавилон и ранний бронзовый век

- открыта вторая эпоха — «Царства бронзового века»;
- полноценная кампания «Вавилон и аморейские царства»;
- 8 глав, 48 миссий и 96 карточек;
- Исин, Ларса, Мари, Вавилон, Хаммурапи, судебные документы и староассирийская торговля;
- карта кампании и обзор шести регионов бронзового века;
- параллельная хронология Месопотамии, Египта, Анатолии, Эгейского мира и Леванта;
- итоговый экзамен кампании из четырёх модулей;
- 8 архивных пулов, 8 личных историй и отдельный пак;
- старые сохранения продолжают использовать ключ `codex_history_v02_ru`.
