# Codex of History v1.6.0

Статическое приложение для GitHub Pages.

## Patch v1.6

- карточки загружают уменьшенные изображения Wikimedia шириной до 900 px;
- Service Worker сохраняет изображения локально в Cache Storage после первой загрузки;
- удалён блок активной кампании с главной;
- счётчик фрагментов в паках исправлен для телефона;
- точки главы фокусируют и приближают карту, не открывая карточку;
- игровой прогресс v1.5 сохраняется.

## Локальный запуск

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть: `http://localhost:8000`

`index.html` двойным кликом открывать нельзя: данные загружаются через `fetch`.
