# Codex of History v1.5

Статическая историческая игра для GitHub Pages.

## Главное в v1.5

- компактная карточка без пустых огромных блоков;
- источники карточек из русской Wikipedia;
- карта главы с реальными кликабельными точками;
- отдельная вкладка паков с собственными обложками;
- уплотнённые миссии, профиль и мобильный интерфейс;
- настройки содержат только тему, обновление и перенос сохранения.

## Локальный запуск

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть `http://localhost:8000`.
