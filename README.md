# Codex of History v2.6.0

## New Player Journey

Новый профиль начинает с красивого семишагового обучения: устройство приложения, миссии, карточки, паки, ежедневное повторение, выбор эпохи и выбор первой кампании. Существующие сохранения не прерываются. Обучение можно запустить повторно из настроек.

## Египет II: фараоны и пирамиды

Патч завершает 10-главную кампанию Египта и добавляет современный учебный интерфейс с выбором эпохи.

- 10 египетских глав
- 60 миссий
- 160 египетских карточек
- 60 уроков с рассказом, хронологией, разбором, теорией и практикой
- новый экран выбора эпохи и кампании
- новый focus-mode обучения
- старые сохранения совместимы

Локальный запуск: `python -m http.server 8000`.
