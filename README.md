# Codex of History v3.1.2

Статическое историческое приложение для GitHub Pages. Контент хранится в JSON, прогресс — в `localStorage`.


## v3.1.2 — исторический визуальный архив

- для всех 899 карточек добавлены поисковые профили изображений;
- приложение подбирает изображения через Wikipedia PageImages и получает автора, лицензию и страницу файла через Wikimedia Commons ImageInfo;
- 42 вручную проверенных изображения Рима остаются статическими;
- остальные карточки получают реальные фотографии артефактов, памятников, карт и исторических изображений при первом подключении к сети;
- найденные данные сохраняются в браузере, сами изображения кэшируются Service Worker;
- при ошибке сети остаётся отдельная локальная SVG-обложка каждой карточки;
- в настройках добавлены состояние визуального архива, повторное обновление и очистка кэша;
- старт новой игры по-прежнему выдаёт карты только выбранной кампании.
