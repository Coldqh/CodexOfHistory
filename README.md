# Codex of History v2.0.0

Статическая историческая обучающая игра для GitHub Pages.

## v2.0

- три главы кампании Рима;
- 130 карточек;
- 31 учебная миссия;
- 31 теория;
- 99 исторических связей;
- 16 архивных пулов;
- 18 личных историй;
- интерактивная карта, квизы, паки, коллекция и интервальные повторения.

### Глава III — Борьба за Италию

12 уроков: Вейи, галльское разграбление, восстановление, Латинская война, союзная система, Самнитские войны, Кавдинское ущелье, дороги и колонии, Сентин, Тарент, Пирр и итоговая модель римского господства.

## Локальный запуск

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть `http://localhost:8000`.
