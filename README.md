# Codex of History v1.9.0

Статическое игровое приложение по истории для GitHub Pages.

## Patch v1.9 — Encyclopedic Theory Layer

Каждая из 19 миссий теперь состоит из пяти учебных этапов:

```text
Рассказ → Хронология → Разбор → Теория → Практика
```

### Теоретический слой

- 19 отдельных нейтральных статей;
- около 6 900 слов полного чтения;
- связное изложение без игрового тона;
- исторический контекст, ход событий, институты и последствия;
- отдельные оговорки о легендарных и спорных сведениях;
- ссылки на статьи Википедии для дальнейшего чтения;
- прогресс чтения и статус после 85% статьи;
- сохранение позиции чтения;
- изменение размера шрифта внутри режима теории;
- прохождение миссии не блокируется непрочитанной теорией.

### Полная структура урока

1. короткий исторический рассказ;
2. хронология;
3. понятия, причины и последствия;
4. длинная нейтральная теория;
5. практика, карта, хронология или итоговый тест.

### Контент

- 2 главы Рима;
- 19 миссий;
- 19 учебных уроков;
- 19 теоретических статей;
- 80 карточек;
- 79 исторических связей;
- 13 личных историй.
