# Codex of History v2.3.0

Полностью завершённая кампания **«Месопотамия: первые города»**.

## Содержание

- 10 глав Месопотамии;
- 60 миссий;
- 160 месопотамских карточек;
- письмо, шумерские города, ранние династии, Аккад и Ур III;
- 10 архивных пулов;
- 13 персональных микроисторий Месопотамии;
- локальные SVG-обложки;
- полный финальный учебный цикл.

## Запуск

```powershell
python -m http.server 8000
```

Открыть `http://localhost:8000`.
