## v3.2.1 — сессионная загрузка изображений

- Исторические изображения карточек загружаются только для карточек, которые реально видит игрок.
- Найденные изображения хранятся только до конца текущей сессии приложения.
- После закрытия приложения они удаляются, а при новой сессии загружаются заново по мере просмотра.
- Service Worker больше не сохраняет удалённые изображения карточек в постоянный кэш.

Статическое историческое приложение для GitHub Pages. Контент хранится в JSON, прогресс — в `localStorage`.
