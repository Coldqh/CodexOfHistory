# Codex of History v2.1.0

## Global Era System

Патч переводит проект от одной римской кампании к глобальной истории древнего мира.

### Добавлено

- 7 глобальных эпох;
- каталог из 30 запланированных кампаний;
- экран эпох, кампаний и мировой хронологии;
- 250 глав в общем плане;
- Рим встроен в эпоху эллинистического и римского мира;
- Месопотамия отмечена как следующая кампания;
- все старые сохранения Рима остаются действующими.

### Запуск

```bash
python -m http.server 8000
```

Открой `http://localhost:8000`.
