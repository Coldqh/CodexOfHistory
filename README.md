# Codex of History v3.0.0

Статическое историческое приложение для GitHub Pages. Контент хранится в JSON, прогресс — в `localStorage`.

## v3.0 — Рождение цивилизаций: общий слой эпохи

- новая кампания «Первые цивилизации: общий мир»;
- 6 глав, 30 миссий и 60 карточек;
- глобальная карта Месопотамии, Египта, Инда и раннего Китая;
- параллельная хронология;
- экзамен эпохи из 6 модулей;
- общий архивный пак эпохи;
- локальные изображения для всех карточек;
- исправлена смена карт между кампаниями;
- принудительное обновление оставлено только в настройках.

## Локальный запуск

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть: `http://localhost:8000`
