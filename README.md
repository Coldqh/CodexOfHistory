# Codex of History — v0.6

Статичное историческое приложение для GitHub Pages.

## Главный модуль

**Rome Chapter One: Birth of Rome**

- 7 миссий;
- карточки раннего Рима;
- квизы;
- карта;
- хронология;
- финальный экзамен;
- ночная и пергаментная темы.

## Запуск

Откройте `index.html` или запустите:

```powershell
python -m http.server 8000
```

Для GitHub Pages держите `index.html` в корне репозитория.

## Данные

- `data/campaign_rome_ru.json`
