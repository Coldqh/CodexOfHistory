# Codex of History — v0.9

Статическое игровое приложение по истории для GitHub Pages.

## Что есть в версии

- первая кампания «Рождение Рима»;
- сюжетные и архивные карточки;
- архивные пулы, открываемые миссиями;
- паки только из пройденных этапов активной кампании;
- защита от спойлеров будущих глав и эпох;
- 42 карточки;
- 6 личных историй персонажей;
- мастерство карточек, фрагменты и крафт;
- интерактивная карта Leaflet;
- чёрная и пергаментная темы;
- мобильный и настольный интерфейсы;
- сохранение прогресса через `localStorage`.

## Главные файлы

- `index.html`
- `styles.css`
- `app.js`
- `patch-v08.js`
- `patch-v09.js`
- `data/campaign_pools.json`
- `data/personal_stories.json`
- `data/pack_rules_v09.json`

## Локальный запуск

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть `http://localhost:8000`.

## GitHub Pages

Публикация из ветки `main`, папка `/root`.

## Внешние ресурсы

Карта использует Leaflet и тайлы OpenStreetMap. Изображения загружаются через Wikimedia Commons. При ошибке включается локальный fallback.
