# Codex of History v1.8.0

Статическое игровое приложение по истории для GitHub Pages.

## Patch v1.8 — Learning Mission System

Миссии больше не состоят из команды «открой карточки» и отдельного теста. Каждая миссия — полноценный короткий урок:

```text
Рассказ → Хронология → Разбор → Практика
```

### Внутри каждой миссии

- связное историческое объяснение;
- учебные цели;
- собственная хронология;
- отметка традиционных и приблизительных дат;
- термины;
- причины и последствия;
- работа с источником, карта, система институтов или другая практика;
- сворачиваемый архив карточек.

### Контент

- 2 главы Рима;
- 19 миссий;
- 19 учебных уроков;
- 80 карточек;
- 79 исторических связей;
- 13 личных историй;
- редкости: 39 Rare / 29 Epic / 11 Legendary / 1 Mythic.

## Запуск локально

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть:

```text
http://localhost:8000
```

## Проверка

```powershell
npm test
```

Валидатор не пропустит миссию без рассказа, хронологии, понятий, причин, последствий или практики.
