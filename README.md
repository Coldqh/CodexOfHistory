# Codex of History v1.4

Статическая историческая игра для GitHub Pages. Версия 1.4 чистит мобильный интерфейс, заменяет нижнюю панель боковым меню, уплотняет архив, коллекцию и паки, исправляет safe-area на iPhone и приводит отображение версии к одному значению.

## Главное

- мобильное боковое меню вместо нижней панели;
- безопасная цветная зона под статус-баром iPhone;
- компактные архив, коллекция и фильтры;
- архив показывает только полученные карточки;
- удалён крупный блок освоения из библиотеки;
- компактные паки в одну горизонтальную линию;
- возвращены изображения паков;
- очищен профиль, сброс прогресса перенесён вниз;
- убран описательный блок поверх основной карты;
- убран стандартный префикс Leaflet из атрибуции;
- версия приложения — `v1.4.0`.

## Установка

Распакуйте содержимое архива в корень `C:\CodexOfHistory`, затем:

```powershell
git add .
git commit -m "Patch v1.4 mobile cleanup and navigation"
git push
```

## Локальный запуск

```powershell
cd C:\CodexOfHistory
python -m http.server 8000
```

Открыть `http://localhost:8000`.

## Проверки

```powershell
npm test
```
