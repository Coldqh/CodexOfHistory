# Codex of History v3.1.3

Статическое историческое приложение для GitHub Pages. Контент хранится в JSON, прогресс — в `localStorage`.


## v3.1.3 — проверка смысла изображений

- пересобраны профили всех 899 карточек;
- случайные резервные темы удалены;
- 80 неоднозначных названий получили ручные исторические развязки;
- 307 рискованных тем проверяются в строгом режиме;
- статья проверяется по описанию, вводному тексту, категориям, региону и типу карточки;
- страницы о животных, фильмах, песнях, компаниях и других одноимённых темах отклоняются;
- при низкой уверенности остаётся локальная SVG-обложка;
- старый кэш ошибочных совпадений v3.1.2 удаляется автоматически;
- старт новой игры по-прежнему выдаёт карты только выбранной кампании.
