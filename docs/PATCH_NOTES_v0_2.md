# PATCH v0.2 — RU UI / Desktop / Mobile / Images

## Сделано

- Полная русификация MVP-интерфейса.
- Переписаны 14 стартовых карточек Рима на русском.
- Переписаны кампания и квизы.
- Добавлена новая дизайн-система: тёмный codex, золото, бордовый, карта, карточный стиль.
- Добавлены два интерфейсных режима:
  - desktop: боковая навигация, широкие сетки, split-view карточки;
  - mobile: верхняя панель, нижняя навигация, крупные карточки.
- Для каждой карточки подключено изображение из Wikimedia Commons.
- Добавлен `ATTRIBUTION.md` и `data/image_manifest.json`.
- Добавлен fallback `assets/ui/fallback-card.svg`.
- Обновлена структура данных для будущего слоя media assets.

## Ограничения

- Изображения подключаются внешними ссылками через Wikimedia Commons `Special:FilePath`.
- Перед публичной монетизацией нужно вручную пройтись по каждой странице файла и проверить лицензию.
- Backend всё ещё отсутствует. Прогресс хранится в `localStorage`.
