# Codex of History — Patch v0.4

## Главный апгрейд

Полностью переработан UI/UX без изменения статической архитектуры GitHub Pages.

### Новый визуальный язык

- cinematic historical codex;
- глубокий графитовый фон, золото, бордо и холодный синий;
- editorial-типографика;
- стеклянные слои, мягкий свет и текстурный фон;
- единая визуальная иерархия вместо одинаковых панелей.

### Главная

- новая hero-сцена без фотографии текущей карточки;
- анимированный круг прогресса;
- компактная статистика;
- действия встроены в hero без блоков «Быстрый доступ» и «Сейчас в кампании».

### Кампания

- cinematic header;
- круговой индикатор прогресса;
- полноценный вертикальный маршрут;
- визуальные состояния: закрыто, доступно, завершено;
- сохранены проценты и проходной порог квизов.

### Коллекция

- новый карточный дизайн;
- 3D tilt на ПК;
- подсветка курсором;
- плавный zoom изображений;
- sticky-панель поиска и фильтров;
- отдельная мобильная сетка.

### Карточка

- большой cinematic cover;
- двухколоночный desktop-layout;
- отдельные блоки сути и значения;
- нумерованные факты;
- новый блок связей;
- sticky-панель характеристик.

### Квизы

- focus mode;
- новый счётчик вопросов;
- анимированные ответы;
- экран результата;

### Карта

- стилизованный средиземноморский слой;

### Профиль
