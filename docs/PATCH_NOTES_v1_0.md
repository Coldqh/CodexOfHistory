# Codex of History v1.0

## Главные изменения

- Архив и полный каталог разделены.
- Архив показывает только полученные карточки.
- Коллекция показывает все 42 карточки, включая будущие этапы.
- Закрытые карточки в каталоге показывают способ получения, но скрывают факты и связи.
- Исправлены изображения 15 архивных карточек.
- Добавлены фильтры каталога по доступности, типу и редкости.
- Добавлена миграция старого прогресса v0.9 без сброса данных.

## Логика

- `ARCHIVE`: только `isUnlocked(cardId) === true`.
- `CATALOG`: все карточки из `CARDS`.
- Сюжетные карточки по-прежнему не выпадают из паков.
- Архивные карточки будущих пулов по-прежнему не могут выпасть раньше времени.
