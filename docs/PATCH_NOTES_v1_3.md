# Patch Notes v1.3 — Compact Mobile & Settings

## Мобильный интерфейс

- Значительно уменьшены hero-блоки, карточки, отступы, кнопки и заголовки.
- Коллекция и архив используют сетку из двух карточек даже на узких телефонах.
- Нижняя навигация стала ниже: неактивные разделы показывают только иконки, активный — иконку и подпись.
- Шапка телефона уменьшена до 50 px и содержит быстрые кнопки темы, обновления и настроек.
- Уплотнены кампания, миссии, квизы, ежедневная сессия, карта и профиль.
- Добавлена комфортная плотность для пользователей, которым нужен более крупный интерфейс.

## Настройки

Новый экран позволяет менять:

- тему;
- плотность мобильного интерфейса;
- размер текста;
- количество анимаций.

Настройки хранятся отдельно от игрового прогресса.

## Обновление

- В интерфейсе отображается версия приложения.
- Добавлена кнопка принудительного обновления.
- Она удаляет Cache Storage, отменяет старые Service Worker-регистрации и перезагружает ресурсы с cache-busting токеном.
- Игровой прогресс при обновлении не удаляется.
- Bootstrap загружает JSON и модули с номером версии и `cache: no-store`.

## Сохранения

- Экспорт прогресса и настроек в JSON.
- Импорт сохранения из JSON.
- Отдельный сброс настроек интерфейса.
- Полный сброс игрового прогресса остаётся отдельным опасным действием.
