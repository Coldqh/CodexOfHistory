# Patch v1.6 — Local Image Cache & Map Focus

- Карточки используют уменьшенные изображения Wikimedia (900 px), а Service Worker сохраняет их в локальный Cache Storage после первой загрузки.
- Удалён блок активной кампании с главной.
- Счётчик фрагментов на экране паков уплотнён и перенесён влево.
- Нажатие на точку главы теперь фокусирует и приближает карту, а не открывает карточку.
- Версия приложения: v1.6.0.
