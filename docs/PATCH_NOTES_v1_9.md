# Patch v1.9 — Encyclopedic Theory Layer

## Цель

Добавить в каждую миссию отдельный академический слой чтения, который не заменяет короткое объяснение и практику, а даёт последовательное подробное изложение темы.

## Изменения

- добавлен пятый этап `Теория`;
- практика перенесена на этап 5;
- добавлено 19 теоретических статей;
- общий объём — около 6 900 слов;
- добавлен полноэкранный прокручиваемый режим чтения;
- добавлен прогресс чтения;
- статус `Прочитано` устанавливается после 85%;
- прогресс сохраняется в `localStorage`;
- добавлена настройка размера текста внутри статьи;
- добавлены исторические оговорки;
- добавлены источники и ссылки для дальнейшего чтения;
- расширена JSON-схема урока;
- валидатор проверяет наличие и объём теории;
- добавлен `smoke-v19-theory.mjs`.

## Структура theory

```json
{
  "theory": {
    "title": "...",
    "readingMinutes": 4,
    "lead": "...",
    "paragraphs": ["..."],
    "historicityNotes": ["..."],
    "sources": [{"title": "...", "url": "..."}],
    "license": "...",
    "checkedAt": "2026-07-10"
  }
}
```

## Совместимость

- старые миссии и сохранения продолжают работать;
- завершённые миссии сразу получают доступ ко всем пяти этапам;
- теория не является обязательным блокиратором прохождения;
- GitHub Pages и полностью статическая архитектура сохранены.
