# Patch v3.1.2 — исторический визуальный архив

- 899 поисковых профилей карточек;
- Wikipedia PageImages для выбора тематического изображения;
- Wikimedia Commons ImageInfo/extmetadata для автора, лицензии и страницы файла;
- браузерный кэш найденных метаданных;
- Service Worker кэширует удалённые миниатюры;
- локальные SVG остаются офлайн-резервом;
- настройки показывают заполнение визуального архива;
- проверка распределения резервных тем и отсутствующих fallback-файлов.
