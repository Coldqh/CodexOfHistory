# QA v3.1

## Автоматические проверки

- `node --check` для всех файлов в `js/`, Service Worker и тестов;
- content validator;
- static smoke v3.1;
- runtime DOM-test v3.1;
- локальная HTTP-проверка главных файлов;
- повторный запуск тестов из распакованного релизного ZIP.

## Проверяемые ограничения

- версия `3.1.0` едина;
- ключ `localStorage` не изменён;
- 8 кампаний, 343 миссии и 343 урока;
- 899 карточек и 71 архивный пул;
- Вавилон: 8 глав, 48 миссий, 96 карточек;
- 64 сюжетные и 32 архивные карты;
- 12 квизов, 68 уникальных вопросов;
- правильные ответы распределены по четырём позициям поровну;
- 8 личных историй;
- 4 модуля итогового экзамена;
- 6 регионов и 5 периодов параллельной хронологии;
- у каждой карточки отдельный существующий локальный файл изображения;
- редкостная пирамида строго убывает;
- принудительное обновление доступно только в настройках.

## Ограничение среды

Chromium установлен, но открытие локального HTTP-адреса блокируется политикой среды `ERR_BLOCKED_BY_ADMINISTRATOR`. Поэтому браузерная проверка дополнена runtime DOM-тестом, статической проверкой адаптивных CSS-правил и HTTP-проверкой файлов.
