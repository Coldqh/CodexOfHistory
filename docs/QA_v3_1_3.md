## Правила приёмки

1. Статья должна соответствовать историческому региону карточки.
2. Для рек, городов, людей, народов, государств и памятников проверяется тип темы.
3. Страницы о животных, фильмах, песнях, играх, компаниях и спортивных клубах отклоняются.
4. Контекстный fallback может использоваться максимум для восьми карточек.
5. Низкая уверенность означает локальную обложку, а не случайное изображение.
6. Кэш v3.1.2 не используется.

## Контрольные случаи

- «Тигр» + страница о Panthera tigris → отклонено;
- «Тигр» + страница реки Месопотамии → принято;
- «Коза» + страница домашнего животного → отклонено;
- «Коза» + страница древнеримской колонии Cosa → принято.
